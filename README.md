<h1 align="center"> UniEventos Frontend 🎉 </h1>

<p>¡Bienvenido al repositorio del frontend de UniEventos! Esta aplicación permite a los usuarios adquirir entradas para conciertos y eventos en diversas ciudades de Colombia. La interfaz de usuario está desarrollada con <strong>Angular</strong> para ofrecer una experiencia rápida y dinámica.</p>

<h2>🚀 Tecnologías Utilizadas</h2>
<ul>
    <li><strong>Angular</strong>: Framework principal para construir la interfaz.</li>
    <li><strong>TypeScript</strong>: Lenguaje principal del desarrollo.</li>
    <li><strong>HTML</strong> y <strong>CSS</strong>: Para la estructura y estilo de la aplicación.</li>
</ul>
