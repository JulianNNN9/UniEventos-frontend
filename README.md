<h1 align="center"> UniEventos Frontend 🎉 </h1>
¡Bienvenido al repositorio del frontend de UniEventos! Esta aplicación permite a los usuarios adquirir entradas para conciertos y eventos en diversas ciudades de Colombia. La interfaz de usuario está desarrollada con Angular para ofrecer una experiencia rápida y dinámica.

##🚀 Tecnologías Utilizadas
*Angular: Framework principal para construir la interfaz.
*TypeScript: Lenguaje principal del desarrollo.
*HTML y CSS: Para la estructura y estilo de la aplicación.
